# HW 4 Кеширование

### Общая инфомация
Используется Reddis в качестве БД хранения Кеша

В кеше хранится 1000 постов (настраивается через `FEED_CACHE_SIZE`)


### Инвалидация кеша

#### 1 При удалении/создании/обновлении поста поста
- Инвалидирует кеш всех друзей

#### 2 При отписке/добавлении от друга
- Инвалидирует кеш пользователя и друга

#### 3 По времени жизни (TTL)
- Автоматическая инвалидация через `FEED_CACHE_TTL` секунд


### Перестройка кешей из СУБД.
Endpoint /post/feed/rebuild -  перстраивает кеш для задданного пользователя(или пользователя под которым залогинился)


### Добавленные файлы по работе с кешем
1. **app/core/redis_client.py** - менеджер подключения к Redis
2. **app/services/cache_service.py** - сервис для работы с кешем
